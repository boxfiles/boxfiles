# @boxfiles/provider-gpu

GPU context provider placeholder for Boxfiles.

This package reserves the `gpu` plugin capability for future workstation facts about graphics hardware, drivers, and GPU-related package setup.

Expected future facts may include GPU vendor, driver stack, acceleration availability, and platform-specific capabilities.

## Status

No actions or context resolvers are implemented yet. The package currently registers an empty `gpu` plugin so manifests and package boundaries can stabilize before implementation.
